# 基金定投分析网站

## 项目背景

一个面向普通投资者的基金定投分析 Web 应用，帮助用户：
- 搜索和了解基金基本信息
- 模拟定投策略并分析收益
- 管理自选基金列表
- 可视化展示基金走势和定投效果

## 本地启动方式

### 方法一：直接运行

```bash
cd src
pip install -r ../requirements.txt
python main.py
```

### 方法二：使用部署脚本

**Windows:**
```powershell
powershell -ExecutionPolicy Bypass -File deploy.ps1
```

**Linux/Mac:**
```bash
chmod +x deploy.sh
bash deploy.sh
```

### 方法三：Docker 部署

```bash
docker-compose up -d
```

然后在浏览器打开 `http://localhost:8000`

## 技术栈

- **后端**: Python 3.10 + FastAPI
- **前端**: Jinja2 模板 + Alpine.js + ECharts
- **数据库**: SQLite (SQLAlchemy ORM)
- **数据源**: AKShare
- **认证**: Session 认证（简单用户名密码）
