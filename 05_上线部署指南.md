# 基金定投分析网站 - 上线部署指南

## 一、上线前准备工作

### 1.1 安全加固

1. **修改密钥**
   - 打开 `src/config.py`，修改 `SECRET_KEY` 为强随机密钥
   - 打开 `src/main.py`，修改 SessionMiddleware 的 secret_key

2. **密码加密升级**
   - 当前使用简单的 SHA256，建议升级为 bcrypt
   - 安装：`pip install bcrypt`
   - 修改 `src/routes/auth.py` 中的 hash_password 函数

3. **数据库备份**
   - 定期备份 `data/fund.db` 文件
   - 生产环境建议使用 PostgreSQL 替代 SQLite

### 1.2 性能优化

1. **添加缓存**
   ```python
   # 在 recommend_service.py 中添加缓存
   from functools import lru_cache
   
   @lru_cache(maxsize=128)
   def get_cached_recommend_funds():
       return get_recommend_funds()
   ```

2. **异步优化**
   - 考虑使用 asyncio 提高并发处理能力
   - 添加请求限流防止恶意调用

### 1.3 监控日志

1. **添加日志文件**
   ```python
   # 在 main.py 中添加
   import logging
   logging.basicConfig(
       level=logging.INFO,
       handlers=[
           logging.FileHandler('app.log'),
           logging.StreamHandler()
       ]
   )
   ```

## 二、部署方案选择

### 方案 A：云服务器部署（推荐）

#### 2.1 阿里云 ECS 部署步骤

1. **购买服务器**
   - 推荐配置：2核4G，50GB SSD
   - 操作系统：Ubuntu 22.04 LTS
   - 带宽：3-5Mbps

2. **环境准备**
   ```bash
   # 更新系统
   sudo apt update && sudo apt upgrade -y
   
   # 安装 Python 3.10+
   sudo apt install python3.10 python3.10-venv python3-pip -y
   
   # 安装 Nginx
   sudo apt install nginx -y
   ```

3. **部署应用**
   ```bash
   # 创建项目目录
   mkdir -p ~/fund-analysis
   cd ~/fund-analysis
   
   # 上传项目文件
   # 使用 scp 或 Git 克隆
   
   # 创建虚拟环境
   python3.10 -m venv venv
   source venv/bin/activate
   
   # 安装依赖
   pip install -r requirements.txt
   
   # 创建 systemd 服务
   sudo nano /etc/systemd/system/fund-analysis.service
   ```

4. **systemd 服务配置**
   ```ini
   [Unit]
   Description=Fund Analysis Service
   After=network.target
   
   [Service]
   Type=simple
   User=ubuntu
   WorkingDirectory=/home/ubuntu/fund-analysis/src
   ExecStart=/home/ubuntu/fund-analysis/venv/bin/python -m uvicorn main:app --host 127.0.0.1 --port 8000
   Restart=always
   RestartSec=10
   
   [Install]
   WantedBy=multi-user.target
   ```

5. **Nginx 反向代理**
   ```nginx
   server {
       listen 80;
       server_name your-domain.com;
       
       location / {
           proxy_pass http://127.0.0.1:8000;
           proxy_set_header Host $host;
           proxy_set_header X-Real-IP $remote_addr;
           proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
           proxy_set_header X-Forwarded-Proto $scheme;
       }
       
       location /static/ {
           alias /home/ubuntu/fund-analysis/src/static/;
           expires 30d;
       }
   }
   ```

6. **SSL 证书配置**
   ```bash
   sudo apt install certbot python3-certbot-nginx -y
   sudo certbot --nginx -d your-domain.com
   ```

### 方案 C：本地服务器部署

#### 4.1 Docker 容器化部署

1. **创建 Dockerfile**
   ```dockerfile
   FROM python:3.10-slim
   
   WORKDIR /app
   
   COPY requirements.txt .
   RUN pip install --no-cache-dir -r requirements.txt
   
   COPY src/ ./
   
   EXPOSE 8000
   
   CMD ["uvicorn", "main:app", "--host", "0.0.0.0", "--port", "8000"]
   ```

2. **构建镜像**
   ```bash
   docker build -t fund-analysis .
   docker run -d -p 8000:8000 --name fund-app fund-analysis
   ```

3. **Docker Compose**
   ```yaml
   version: '3.8'
   services:
     web:
       build: .
       ports:
         - "8000:8000"
       volumes:
         - ./data:/app/data
       restart: always
   ```

## 三、域名备案（中国大陆）

### 5.1 备案流程

1. **准备材料**
   - 个人身份证
   - 域名证书
   - 服务器购买凭证

2. **提交备案**
   - 通过阿里云/腾讯云控制台提交
   - 等待审核（通常 1-20 个工作日）

3. **ICP 许可证**
   - 个人博客类网站：只需 ICP 备案
   - 商业网站：可能需要 ICP 许可证

### 5.2 域名配置

1. **域名解析**
   ```
   A 记录：your-domain.com -> 服务器 IP
   www 记录：www.your-domain.com -> 服务器 IP
   ```

2. **HTTPS 配置**
   - 使用 Let's Encrypt 免费证书
   - 配置自动续期

## 七、快速部署命令

### 一键部署脚本

```bash
#!/bin/bash
# deploy.sh

# 更新系统
sudo apt update && sudo apt upgrade -y

# 安装依赖
sudo apt install python3.10 python3.10-venv python3-pip nginx certbot -y

# 创建项目目录
mkdir -p ~/fund-analysis
cd ~/fund-analysis

# 部署应用
git clone <your-repo-url> .
python3.10 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# 配置服务
sudo cp fund-analysis.service /etc/systemd/system/
sudo systemctl daemon-reload
sudo systemctl enable fund-analysis
sudo systemctl start fund-analysis

# 配置 Nginx
sudo cp nginx.conf /etc/nginx/sites-available/fund-analysis
sudo ln -s /etc/nginx/sites-available/fund-analysis /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl reload nginx

# 配置 SSL
sudo certbot --nginx -d your-domain.com

echo "部署完成！"
```
